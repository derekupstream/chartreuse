# Running

1. Retrieve [environment variables](https://vercel.com/upstreamsolutions/upstream-calculator/settings/environment-variables) from Vercel.
2. Add them to a `.env` file in the folder root.
3. Install dependencies:
```bash
yarn install
```
4. Run the development server:

```bash
yarn dev
```
